== Plugin Setup  ==
Open plugin menu and Add plugin
Click on upload plugin and upload plugin folder
After Uploding Plugin Active it

== About  shortcode ==
Here is example of shortcode

[cta_box title="Start Using Divi" message="Discover the power and flexibility of Divi, the ultimate WordPress theme and visual page builder. Create stunning websites with ease and elevate your web design game. Click the button below to get started with Divi today!" button_label="Try it for free" button_url="https://elegantthemes.com"]


Title, Message, Button label and Button url is a custom. You can change button url, button label, message etc.

Using this shortcode you can redirect to your destination page.

On button click it's count number of button click. Count value store in localstorage.
